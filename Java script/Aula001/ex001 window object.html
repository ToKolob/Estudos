Window.####

No JavaScript, o objeto window representa a janela do navegador e fornece acesso a várias funcionalidades e propriedades relacionadas à janela e ao navegador.

  1. alert(): Exibe uma caixa de diálogo com uma mensagem para o usuário.
  2. confirm(): Exibe uma caixa de diálogo com uma mensagem e botões de confirmação ("OK" e "Cancelar"). Retorna true se o usuário clicar em "OK" e false se o usuário clicar em "Cancelar".
  3. prompt(): Exibe uma caixa de diálogo com uma mensagem e um campo de entrada de texto. Retorna o valor inserido pelo usuário ou null se o usuário cancelar a operação.
  4. open(): Abre uma nova janela do navegador ou uma janela pop-up.
  5. close(): Fecha a janela atual.

Também possui muitas propriedades úteis, como location, document, navigator, entre outras, que fornecem acesso a informações e recursos relacionados ao navegador.
