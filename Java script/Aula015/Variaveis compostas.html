Variáveis compostas no JavaScript

Arrays

Um array é uma coleção ordenada de elementos, onde cada elemento é identificado por um índice numérico. Os arrays em JavaScript podem conter valores de qualquer tipo, como números, strings, objetos e até mesmo outros arrays. Os elementos de um array são armazenados sequencialmente, começando do índice 0.

Exemplo de declaração e uso de um array:

      
  
  // Declaração de um array vazio
  let numeros = [];
  
  
  // Adicionando elementos ao array
  numeros.push(1);
  numeros.push(2);
  numeros.push(3);
  
  // Acessando elementos do array
  console.log(numeros[0]); // Saída: 1
  console.log(numeros[1]); // Saída: 2
  console.log(numeros[2]); // Saída: 3
  
  // Alterando um elemento do array
  numeros[1] = 4;
  console.log(numeros); // Saída: [1, 4, 3]

Objetos

Um objeto é uma coleção de pares chave-valor, onde cada chave é uma string que identifica um valor associado. Diferentemente dos arrays, os objetos não possuem uma ordem específica para seus elementos. Os valores associados podem ser de qualquer tipo, incluindo números, strings, booleanos, arrays e até mesmo outros objetos.

Exemplo de declaração e uso de um objeto:

      
  // Declaração de um objeto vazio
  let pessoa = {};
  
  // Adicionando propriedades ao objeto
  pessoa.nome = 'João';
  pessoa.idade = 25;
  pessoa.cidade = 'São Paulo';
  
  // Acessando propriedades do objeto
  console.log(pessoa.nome);   // Saída: 'João'
  console.log(pessoa.idade);  // Saída: 25
  console.log(pessoa.cidade); // Saída: 'São Paulo'
  
  // Alterando uma propriedade do objeto
  pessoa.idade = 26;
  console.log(pessoa); // Saída: { nome: 'João', idade: 26, cidade: 'São Paulo' }
