somando numeo

Explicando

O java script pode ser inserido tanto em um arquivo a parte como dentro do propio corpo do documento, nesse caso foi inserido dentro do corpo do próprio texto

    Vamos aos passos do programa:
  1. Dentro do body HTML colocamos 2 entradas de dados e seus atributos foram type, name e id.
    Os dois primeiros abrem as caixas para o usuario inserir os dados.
  2. Colocamos mais um input dessa vez com o type button que recebeu o values = "somar" e a chamda de função somar() atraves do atributo onclick que chama a função quando o usuario clica do botão.
  3. A div foi criada com id = res.
  4. Dentro do corpo do script criamos um bloco para a função somar
  5. Criamos duas variaves para receberem os valores do usuario atravez da manipulação do DOM.
    Podendo ser feito de diversas formas, por name, id ou querySelector.
  6. Em outras duas variaves convertemos os dados para number.
  7. Criamos um variável para poder alterar pelo DOM a div res.
  8. E a ultima variável para criar calcular a resposta
  9. feito isso a propriedade .innerText para colocar a resposta do script.

Código utilizado:


    <input type="number" name="txtn1" id="txtn1">
    <input type="number" name="txtn2" id="txtn2">
    
    <input type="button" value="somar" onclick="somar()">

    <div id="res"></div>

    <script>

    function somar() {
        var tn1 = window.document.getElementById('txtn1');
        var tn2 = window.document.querySelector('input#txtn2');

        var n1 = Number(tn1.value);
        var n2 = Number(tn2.value);

        var res = window.document.getElementById('res');

        var s = n1 + n2;

        res.innerText = `A soma entre ${n1} e ${n2} é ${s}`;
    }
    </script>
