Imagem dinamica

Para adicionarmos imagens em tamanhos que diferem pela fonte, usamos primeiro a tag picture e em seguida colocamos source:media:type, nessa seguencia escolhemos o termo min ou max-width, e colocamos o tamanho de referência 700px, 1000px, 300px, etc. e ai então usamos a tag img, para cada source.

Tente abrir esse site em varios dispositivos diferentes, ou altere o tamanho do navegador

Se você usar o max-width é crucial que você coloque o source a cima e em ordem do menor para o maior

Reproduzindo áudio

Agora trabalharemos com sons, e para inserir sons no seu site, vocÊ pode acessar a biblioteca de musicas do youtube studio, lá você terá acesso a musicas com licença para utilização.

Usando a tag audio e os parametros controls e o autoplay, seu som deve ter as opções para controle e tocar automaticamente ao iniciar o site (depende do navegador).

os audios que são aceitos são mp3, wav e ogg

Tambem podemos usar o source com o audio, criando diferentes tipos de midia

Vale resaltar que o source tentará carregar os arquivos em ordem, se o primeiro não ler, ele irá para o próximo, até algum que funcione, então da para por uma mensagem de erro personalizada.

Para o audio use source:src no src use ponha o arquivo e em type use audio/mpeg ou ogg ou wav

E dentro do audio use o parametro preload com cuidado para usar o metadata para carregar apenas informações do audio, auto para baixar o audio todo (deixa o site pesado!) e none para só baixar qualquer informação quando o usuario clicar

audio controls preload="metadata"
source src="midia/recording.mpeg" type="audio/mpeg"
source src="audio/recording.wav" type="audio/wav"
infelismente seu navegador não responde aos audios disponíveis, desculpe nossa falha
