Inline

As folhas de Estilo, podem ser aplicadas de 3 formas, a primeira que vamos abordar é a Inline onde o style estará dentro das tags.

Para exemplificar usaremos dentro da tag body o style="blackground-color: lightskyblue;".

A primeira restrição desse metodo é que quando usamos o Inline, a auteração se restringe dentro da tag

Por exemplo vamos aplicar color red no h1, você verá que só o h2 terá a coloração, do mesmo jeito para "text-align: justify; font: normal 20pt Arial;" que coloquei no primeiro parágrafo.
